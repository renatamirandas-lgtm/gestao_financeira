# 🚀 Início Rápido

## 1. Instalar Dependências

```bash
npm install
```

## 2. Executar o Sistema

```bash
npm run dev
```

## 3. Acessar no Navegador

Abra: http://localhost:3000

## 📋 Próximos Passos

1. **Dashboard**: Visualize o resumo financeiro na página inicial
2. **Lançar**: Adicione suas primeiras transações em `/lancar`
3. **Fluxo de Caixa**: Veja o fluxo diário em `/fluxo-caixa`
4. **Resultados**: Analise os resultados mensais em `/resultados`

## 💡 Dica

O sistema está funcionando com armazenamento em memória. Os dados serão mantidos enquanto o servidor estiver rodando.

Para persistência permanente, você precisará implementar um banco de dados (veja README.md para mais detalhes).
